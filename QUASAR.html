Commands

qprefix <new>

Change the bot prefix for your server

qsetad #channel

Set the advertisement channel

qsetrep @role

Set the representative role

qaradd trigger response

Add an auto responder

qsetwelcome #channel

Set welcome message channel

qlb daily/weekly/monthly

View ad leaderboard (premium)

qj2l enable/disable

Toggle J2L auto-ban system

qconfig

View all server settings
