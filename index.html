📜 Privacy Policy

Last Updated: June 15, 2026

1. Information We Collect

QuasarBot collects minimal data required for functionality:

  • Discord User IDs - For economy, pets, and game features
  • Discord Server IDs - For server-specific settings
  • Message IDs - For advertisement tracking features
  • Command Usage - Which commands you use (for cooldowns)

2. How We Use Your Data

  • Economy system (balances, transactions)
  • Pet system (ownership, levels, battles)
  • Server configuration (prefixes, channels)
  • Advertisement and representative tracking
  • Global leaderboards
  • Auto-moderation (J2L system)

3. Data Storage

Data is stored in a secure database. We do not sell, share, or use your data for any purpose other than bot functionality.

4. Data Retention

  • Permanent: Economy balances, pet data, server configs
  • Temporary: Active games, command cooldowns (auto-cleaned)
  • Auto-removed: Ad mentions when users leave servers

5. Your Rights

  • View your data using bot commands
  • Server admins can reset server data
  • Request data deletion by contacting the bot owner

6. Security

We implement encryption, input validation, and rate limiting to protect your data.

Note: By using QuasarBot, you also agree to Discord's Privacy Policy.
