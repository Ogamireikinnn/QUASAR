Privacy Policy

Last Updated: July 8, 2026

1. Introduction

This Privacy Policy explains how QuasarBot collects, uses, and protects information when you use our Discord bot.

2. Information We Collect

Automatically Collected Data

  • Discord User ID - Alliance and representative tracking
  • Discord Server ID - Server-specific settings
  • Message IDs - Advertisement tracking and leaderboard
  • Channel IDs - Ad, log, and welcome channel configuration
  • Role IDs - Representative and bypass role configuration
  • Command Usage - Cooldown management

Data You Provide

  • Custom Prefixes - Set by server administrators
  • Auto Responder Triggers/Responses - Configured per server
  • Welcome Messages - Custom welcome configurations
  • Server Configuration - Ad categories, rep actions, bypass roles

3. How We Use Your Information

  • Bot Functionality - Alliance management, auto responders, welcome messages
  • Server Configuration - Storing custom settings and preferences
  • Advertisement Tracking - Managing rep roles in ad channels
  • Leaderboard System - Ad post statistics for premium servers
  • Auto-Moderation - J2L system for auto-banning
  • Rep Management - Automatic role assignment and removal

4. Data Storage

All data is stored in a secure PostgreSQL/SQLite database.

Permanent Data: Server configurations, auto responder settings, welcome configurations

Leaderboard Data: Stored permanently for premium servers

Advertisement Mentions: Stored until rep leaves or ad is deleted

5. Data Sharing

We do NOT sell, share, or use your data for purposes outside bot functionality.

6. User Rights

Server admins can view config, modify settings, delete data with resetguild, or kick the bot. Bot owner can export, import, or nuke database.

7. Security

Database encryption, secure environment variables, input validation, rate limiting, and permission checks.

8. Data Collected by Command
